# RelationalToNoSQL

Projeto de TCC do curso de Sistemas de Informação da Faculdade Católica do Tocantins.

O sistema tem o intuito de migrar um banco de dados relacional para um banco de dados NoSQL, de forma que o usuário possa selecionar o banco de dados de origem e o de destino. Após selecionar o software deve mapear o banco de dados de origem e mostrar para o usuário selecionar quais tabelas devem ser migradas. Assim o software deve modelar o banco de dados de destino para receber os dados.
